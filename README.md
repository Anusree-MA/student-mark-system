Student Mark Management System

A console-based Python application that helps manage student marks, calculate percentages, and automatically assign grades.

This project is designed for Python beginners to practice loops, conditional statements, and data structures.

---

📌 Features

✔ Add student records
✔ Enter subject marks
✔ Calculate total marks
✔ Calculate percentage
✔ Automatic grade assignment
✔ View all student results

---

🛠 Technologies Used

- Python
- Lists
- Dictionaries
- Loops
- Conditional Statements
